# Customer Churn Analysis Dashboard (Power BI)

## Overview
This Power BI dashboard analyzes SBI customer churn data to identify patterns in customer exits, activity, and retention.

## Features
- Total customers overview
- Active vs inactive customers
- Exit customers analysis
- Credit card holders analysis
- Gender-based churn analysis

## Tools Used
- Power BI
- Data Visualization
